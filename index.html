Brutalist IoT security system with cold precision.

Project Information

This is an IoT home security system built with Python and Flask.

To run locally:

  1. Clone this repository
  2. Install dependencies with pip install -r requirements.txt
  3. Run python src/app.py
  4. Visit http://localhost:5000

GitHub Pages cannot host the full functionality of this application. Please run it locally for the complete experience.
